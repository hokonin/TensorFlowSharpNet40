TFSession Class

Drives the execution of a graph

Syntax

public class TFSession : TFDisposable

Remarks

This creates a new context to execute a TFGraph. You can use the constructo to create an empty session, or you can load an existing model using the FromSAvedModel static method in this class.

Gets a new runner, this provides a simpler API to prepare the inputs to run on a session

Syntax

public TFSession.Runner GetRunner ()

Returns

The runner.

Remarks

The runner has a simple API that allows developers to call the AddTarget, AddInput, AddOutput and Fetch to construct the parameters that will be passed to the TFSession.Run method.
