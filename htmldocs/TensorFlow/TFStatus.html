TFStatus Class

Used to track the result of TensorFlow operations.

Syntax

public class TFStatus : TFDisposable

Remarks

TFStatus is used to track the status of a call to some TensorFlow operations. Instances of this object are passed to various TensorFlow operations and you can use the TFStatus.Ok to quickly check if the operation succeeded, or get more detail from the TFStatus.StatusCode and a human-readable text using the TFStatus.StatusMessage property. The convenience TFStatus.Raise can be used to raise a TensorFlow.TFException if the status of the operation did not succeed.
