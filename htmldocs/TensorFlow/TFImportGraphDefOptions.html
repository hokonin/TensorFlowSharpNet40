AddInputMapping Method

Adds an input mapping from a source name and index to a destination output

Syntax

public void AddInputMapping (string srcName, int srcIndex, TFOutput dst)

Parameters

srcName
Source name.
srcIndex
Source index (in the source).
dst
Replacement value for the srcName:srcIndex.

Remarks

Set any imported nodes with input `src_name:src_index` to have that input replaced with `dst`. `src_name` refers to a node in the graph to be imported, `dst` references a node already existing in the graph being imported into.
