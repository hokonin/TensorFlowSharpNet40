TFStatus Class

Used to track the result of TensorFlow operations.

Syntax

public class TFStatus : TFDisposable

Remarks

TFStatus is used to track the status of a call to some TensorFlow operations. Instances of this object are passed to various TensorFlow operations and you can use the TFStatus.Ok to quickly check if the operation succeeded, or get more detail from the TFStatus.StatusCode and a human-readable text using the TFStatus.StatusMessage property. The convenience TFStatus.Raise can be used to raise a TensorFlow.TFException if the status of the operation did not succeed.

Default Field

Per-thread global status that you can use if you do not need to create a new instance of this object.

Syntax

[System.ThreadStatic]
public static TFStatus Default

Remarks

This is provided as a convenience for APIs that take a TFStatus. While the TFStatus is usually an optional parameter, when it is made optional, API calls that fail raise an exception. Use this property to pass a TFStatus without having to allocate a new one. The problem with this of course is that you risk having multiple parts of your code override this thread-global variable.
